## Project Overview

### What is Pitch Passport?

Pitch Passport is a modern, SEO-optimized football news and analysis platform covering:
- **5 Major Leagues**: Premier League, La Liga, Serie A, MLS, Champions League
- **20 Initial Articles**: High-quality content across all leagues
- **Real-time Data**: Standings, fixtures, and match results
- **Global Audience**: Optimized for search engines and user experience

### Technology Stack

**Frontend**
- Next.js 14 (React framework)
- TypeScript
- Tailwind CSS
- Axios (API client)
- Zustand (state management)

**Backend**
- Node.js + Express.js
- MySQL database
- JWT authentication
- Role-based access control

**Infrastructure**
- Vercel (frontend hosting)
- DigitalOcean/AWS (backend hosting)
- Cloudflare (CDN)
- Google Analytics 4 (analytics)
